# Projet : Suivi pédagogique des TP

## Description
Ce projet est une application Laravel permettant d’afficher la liste des étudiants enregistrés dans la base de données.  
Elle utilise un contrôleur (`EtudiantController`) pour récupérer les données et une vue Blade pour les afficher.

## Technologies
- Laravel
- PHP
- MySQL
